# 题目：二叉树的所有路径

给定一个二叉树，返回所有从根节点到叶子节点的路径。

说明: 叶子节点是指没有子节点的节点。

示例:

输入:

   1
 /   \
2     3
 \
  5

输出: ["1->2->5", "1->3"]

解释: 所有根节点到叶子节点的路径为: 1->2->5, 1->3

# 提交情况：
    -Solution_1 : time:4ms(87.57%)  内存：13.4M(23.60%)
        - 思路: 使用分治方法，某点的路径等于左边每个路径 + 本点  和 右边每个路径 + 本点 的集合。
        - 优化：内存上可以公用一个
